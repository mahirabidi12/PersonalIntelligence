# Blinkit2 - Grocery Delivery Clone

A pixel-perfect clone of Blinkit (India's #1 instant grocery delivery app).

## Project Structure
- **Backend**: `/app/backend/` (FastAPI + MongoDB)
- **Frontend**: `/app/frontend/` (React)

## Features
- 12 categories, 64+ products with INR pricing
- Product browsing with horizontal scroll carousels
- Search products
- Cart with add/remove/quantity controls
- Checkout with address + simulated COD payment
- Order placement & order history
- JWT auth (login/register)
- "10 minutes delivery" branding
